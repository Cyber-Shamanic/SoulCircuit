## דרך האתר

1. צור מאגר חדש בשם `soul-circuit`.
2. אל תוסיף README, רישיון או `.gitignore` — הם כבר קיימים.
3. חלץ את חבילת המאגר לתיקייה מקומית.
4. הרץ את הפקודות:

```bash
git init
git branch -M main
git add .
git commit -m "chore: establish SoulCircuit v0.1.0"
git remote add origin https://github.com/Cyber-Shamanic/soul-circuit.git
git push -u origin main
```

## דרך GitHub CLI

```bash
git init
```
