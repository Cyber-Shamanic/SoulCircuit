# 📚 מאגר המקורות

מטרת התיקייה היא לשמור על דיוק, כבוד וקרדיט. היא מיועדת ללימוד, כתיבה, תפילה, חשבון נפש והתבוננות — לא להוראות קמע, השבעה או פעולה מאגית.

## כללי רישום

1. מעתיקים רק את הקטע הנדרש.
2. רושמים ספר, פרק, פסוק/סעיף ומהדורה.
3. מבדילים בין ציטוט מדויק, עיבוד והשראה.
4. בודקים שהשיר אינו משנה את משמעות המקור.
5. לפני הפצה מעבירים את הרשומה ל־`verified`.

## מקורות בסיס

- תנ״ך: [מכון ממרא](https://mechon-mamre.org/)
- ספרייה יהודית דיגיטלית: [Sefaria](https://www.sefaria.org/)
- פרויקט בן־יהודה: [Ben-Yehuda Project](https://benyehuda.org/)

הקישורים משמשים לאיתור ולקריאה; בכל רשומת שיר נרשם המיקום המדויק שנבדק.
