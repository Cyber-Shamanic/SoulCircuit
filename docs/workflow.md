# 🔄 תהליך עבודה לשיר

## 1. Seed — זרע

פותחים Issue, בוחרים מזהה, כותבים כאב, מסר, דימוי ופאנץ׳ ראשוני.

## 2. Sources — מקורות

רושמים מקור מרכזי, הקשר, ציטוט קצר ומיקום מדויק ב־`sources.yml`.

## 3. Draft — טיוטה

כותבים לפי תבנית השיר, מנקדים, קוראים בקול ומחשבים אורך.

## 4. Lyrics Lock — נעילת מילים

משלימים את 12 שערי האיכות, מקפיאים את גרסת המילים ומעלים גרסת קטלוג.

מגדירים BPM, סולם, חתימת סאונד, עיבוד, הוראות ביצוע ורפרנסים פנימיים.

## 6. Mix & Master

שומרים גרסאות ברורות: `demo`, `rough-mix`, `mix-v1`, `mix-final`, `master`.

## 7. Visual

יוצרים עטיפה, טיפוגרפיה, גרסאות רשת וחומרי קליפ לפי תוכן השיר.

## 8. Release

נועלים קרדיטים, מקורות, זכויות, מטא־דאטה וחבילת ההפצה.
